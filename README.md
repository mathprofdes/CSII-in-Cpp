# CSII in C++

This repository contains course materials and example programs for a CSII course in C++.

**Textbooks Used for the Course:**

- *Starting Out With C++: From Control Structures through Objects by Tony Gaddis*

**Contents:**

- ClassExampleCode_CSI_CSII: Code examples for both CSI & CSII courses.  Most of the CSII code examples contain Linux makefiles.
- Homeworks: A set of homework assignments from the last time I instructed the class.
- Labs: A set of lab assignments from the last time I instructed the class.
- Projects: A set of projects from the last time I instructed the class.
